Supported Components¶

As of this version, HWProbe supports retrieving info from the following components.

  • CPU

  • GPU

  • Memory

  • Network

  • Storage

To get started, visit Quickstart.

Attributes gathered for each component are listed in the component’s schema, in the Models section.
